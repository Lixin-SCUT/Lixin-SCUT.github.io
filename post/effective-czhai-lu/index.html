Effective C++摘录

· 2020-03-16 · # C++

今天遇到一道题：

#include <bits/stdc++.h>
using namespace std;

class A {
public:
    virtual void foo(int a = 0) {
        cout << a << endl;
    }
};

class B : public A {
public:
    virtual void foo(int a = 1) {
        cout << a << endl;
    }
};

int main() {
    A* p = new B();
    p->foo();
}

问输出是啥？
很明显问题在于默认值
根据条款37:绝不重新定义继承而来的缺省参数值，因为缺省参数值是静态的，所以输出为0.
感受到了Effective C++的强大，所以特地整合一下Effective的大概条款，方便以后扩充
