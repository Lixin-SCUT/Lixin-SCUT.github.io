位运算 面试题 01.01. 判定字符是否唯一[简单]

· 2020-04-11 · # leetcode

实现一个算法，确定一个字符串 s 的所有字符是否全都不同。
示例 1：
输入: s = "leetcode"
输出: false
示例 2：
输入: s = "abc"
输出: true
限制：
0 <= len(s) <= 100
如果你不使用额外的数据结构，会很加分。
来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/is-unique-lcci
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

最明显的就是哈希表了吧，先复习一波
然后就是固定长度的字母表了，进一步可以简化为位运算法。
最后就是排序法，就不额外实现了

当然这道题是默认的小写字母，如果加上大写字母或者其他字符，就要更加细心地考虑了。

class Solution {
public:
    // 位运算
    bool isUnique(string astr) 
    {
        int bitMap = 0;
        for(char c : astr)
        {
            int bit = c - 'a';
            if((1 & (bitMap >> bit)) == 1) // 千万记得 == 的优先级高于 相与运算符， 一定要加括号
            {
                return false;
            }
            else
            {
                bitMap = bitMap | (1 << bit);
            }
        }
        return true;
    }
