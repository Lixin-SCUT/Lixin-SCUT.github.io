leetcode链表测试代码

· 2019-11-01 · # leetcode

注意在C++里面ListNode已经是定义好的类了，不需要重复定义了。
本代码主要是建立一个链表方便在playground中进行测试，但是注意输出其返回值的话不会像测试用例那样会输出整个链表。

int main() {
    ListNode n5(5);
    ListNode n4(4);
    ListNode n3(3);
    ListNode n2(2);
    ListNode n1(1);
    ListNode *head=&n1;
    n1.next=&n2;
    n2.next=&n3;
    n3.next=&n4;
    n4.next=&n5;
    cout<<swapPairs(head)->val<<endl;
}
